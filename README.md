# make ufs image

This is how you can set up the environment to make a ufs image. There may be other methods but this is how ive done it.

# Prerequisites

- FreeBSD

# instructions (in bash in freebsd terminal)

- `git clone https://github.com/earthonion/mkufs2/`
- `chmod +x mkufs2.sh`
- `./mkufs2.sh <input_dir> [output_file]`
